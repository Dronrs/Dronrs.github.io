Python Lark Parser使用

 2021-04-26  137 words  parse 

Python Lark库是我最近找到的PLY库的替代品，使用上比PLY不知高到哪去了，今天就来讲讲这个Lark库的简要用法。

首先，使用Lark库需要从pip中安装，命令如下：

pip install lark-parser --upgrade

接着，我来写一个简单的moew语言，然后打印出它的语法树：

import Lark from lark

parser = Lark(r'''
    moewlang: moew ("," moew)*
    moew: "moew"
''', start="moewlang")

print(parser.parse("moew,moew,moew").pretty())

输出：

moewlang
  moew
  moew
  moew
